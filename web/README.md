# NAOS.js

A JavaScript/TypeScript library for interfacing with [NAOS](https://github.com/256dpi/naos) devices.

## Installation

Add to your `package.json`:

```json
"naos": "https://github.com/256dpi/NAOS.js#TAG_OR_COMMIT"
```

## Note

This repository is a mirror of the `web` directory in the main [NAOS](https://github.com/256dpi/naos) repository. Please open issues and pull requests there.
